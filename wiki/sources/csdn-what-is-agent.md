## 为什么值得保留

这篇材料把 Agent 展开成一个四能力模型：

- 感知
- 规划
- 行动
- 记忆

## 关键要点

- 文中把 Agent 和只会聊天的模型做了明确区分。
- 它强调多模态感知、动态规划、工具调用和长期记忆。
- 它把 MCP、RAG 等机制放进了行动能力和记忆能力的语境里。
